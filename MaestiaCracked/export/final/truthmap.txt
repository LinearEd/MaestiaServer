AUTH / LOGIN / SESSION
ID	Dir	Name	Beschreibung	Root Handler	Confidence
0x0200	C→S	LOGIN_REQ	Login Request (Account + Hash)	FUN_00874470
0x0201	S→C	LOGIN_ACK	Login Antwort / Result	FUN_00a422f0
0x0202	C→S	LOGIN_RSA	RSA Key Exchange / Proof	FUN_00a422f0
0x0204	C→S	LOGIN_PING	Login keepalive	FUN_00a39a70
0x0205	S→C	LOGIN_STATUS	Login Status Update	FUN_00a422f0

Erklärung
FUN_00a422f0 ist klar ein zentraler Login-State-Handler.
